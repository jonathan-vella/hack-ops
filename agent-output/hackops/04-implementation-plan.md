## 📋 Overview

This implementation plan defines the Bicep infrastructure for
HackOps — an Azure hackathon management platform. The architecture
deploys a single-region PaaS stack with private networking:

- **Compute**: App Service (Linux, Node 22 LTS) with Easy Auth
- **Database**: Cosmos DB NoSQL (Serverless) with 10 containers
- **Security**: Key Vault with Private Endpoint, managed identity
- **Networking**: VNet with 3 subnets, NSGs, Private DNS Zones
- **Observability**: Log Analytics + Application Insights

**Governance compliance**: 21 policies discovered, 0 deployment
blockers. 9 mandatory tags enforced on resource group (Deny).
Cosmos DB local auth auto-disabled (Modify). See
[04-governance-constraints.md](04-governance-constraints.md).
